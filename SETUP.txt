AUTO EMAIL BACKUP SETUP (optional)
------------------------------------
Backup sends silently to carlorinia@gmail.com every 7 days.

1. Go to https://www.emailjs.com and create a free account
2. Add a Gmail service → copy your Service ID
3. Create an Email Template with these variables:
      To:       {{to_email}}
      Subject:  Lew's Tracker Backup — {{backup_date}}
      Body:
        Date: {{backup_date}}
        Stats: {{backup_stats}}

        --- JSON BACKUP (copy and save as .json) ---
        {{backup_data}}
4. Copy your Public Key, Service ID, Template ID
5. In the tracker: Settings → Auto Email Backup → paste keys
6. Click "Test Send Now" to verify

NODE.JS WORD EXPORT (optional, for .docx format)
--------------------------------------------------
The browser Word export (⬇ Word button) gives you a .doc file that Word opens fine.
For a proper .docx with full Word formatting, use the included Node.js script:

  1. Install Node.js from https://nodejs.org (if not already installed)
  2. In the tracker folder, run: npm install
  3. Export a JSON backup from Settings → Export
  4. Run: node export-to-docx.js your-backup.json
  5. Opens as a formatted Word document with priority-colour rows

MULTI-USER SETUP (v8.2)
------------------------
The tracker now supports multiple users, each with their own data.

First launch:
  Open lews_tracker_v8_2.html → you will see a Sign In screen
  Click "Create Account" → choose a username and password
  Your data is stored separately from every other user

Sharing your setup with another user:
  Settings → Users & Sharing → Download My Data
  Send the downloaded .json file to the other person
  They open their own account → Settings → Import / Load Shared
  They confirm the import → their tracker loads your tasks & templates

Security notes:
  Passwords are hashed with SHA-256 before storage
  Each user's data is isolated in their own browser storage key
  Sessions last 30 days — sign out manually for immediate lock
  This is local browser storage only — not synced to a server
